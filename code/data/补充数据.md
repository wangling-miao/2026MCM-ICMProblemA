\### 2. Peukert 常数 (用于锂离子电池)



虽然 Peukert 定律最初用于铅酸电池，但对锂离子电池依然有效，只是系数不同：



\*   \*\*锂离子电池 Peukert 常数 ($k$)\*\*: 通常在 \*\*1.05 - 1.10\*\* 之间 。 \[pmc.ncbi.nlm.nih](https://pmc.ncbi.nlm.nih.gov/articles/PMC9863025/)

&nbsp;   \*   这比铅酸电池 (1.3-1.4) 小得多，说明锂离子电池的大电流放电性能较好，但仍有容量损失。

\*   \*\*建议\*\*: 在模型中设定 $k \\approx 1.08$，并进行灵敏度分析 (1.05 - 1.15)。
